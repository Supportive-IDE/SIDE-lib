SIDE-lib Basic Usage

The library's two main functions (parse() and feedback()) take a string of source code as input e.g.:


        
feedback(python_str, includeParseOutput = false)

        
...where python_str is a string of Python code. For best results, this should be a complete program rather than a snippet.

        
feedback() returns a JSON object containing basic information to provide feedback on a detected misconception indicator. E.g.:
